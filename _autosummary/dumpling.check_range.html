dumpling.check_range¶

dumpling.check_range(minimum, maximum)[source]¶

return a function to check the value is in the legal range.

Parameters:
  • minimum (numbers.Number) – the lower bound
  • maximum (numbers.Number) – the upper bound
Returns:

function – A function that receives a numeric value and does the checking, and returns the value if it is legal.

Examples

>>> from dumpling import check_range
>>> f = check_range(0, 9)
>>> f('2')
'2'
>>> f('10')
